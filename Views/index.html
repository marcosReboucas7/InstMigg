InstMigg Web Site

Este site é foi feito com o intuito de registrar os cadastros dos clientes que foram migrados e instalados para termos controle sobre os valores das migrações, Updates e instalações.

Nas migrações e instalações feitas, vimos dificuldades de realizar a documentação dos mesmos. Por causa disso, este site tem o intuito de facilitar a documentação.

Neste site, você consegue cadastrar os contratos migrados, atualizados ou instalados, com seus respectivos valores para pararmos de utilizar o wpp como meio de documentação. Nesse site, também teremos pesquisas.

  • Você está na página de Apresentação. Ou seja, essa página tem o intuito de explicar as funcionalidades do site.
  • Na próxima página, você encontrará a Página de Cadastro, onde você irá colocar os clientes que você migrou ou instalou.
  • Na última página, veremos a página de consulta, onde você poderá consultar todos os clientes e migrações divididos por mês.
